Project Details

Project Name: Cricket for Life

Description: This project is a web-based application designed for cricket enthusiasts. It provides a platform to view cricket match schedules, team details, and statistics. Users can also read cricket-related news and updates.

Programming Languages/Tools: HTML, CSS, JavaScript, PHP, MySQL

Features:

  • View upcoming cricket match schedules and details
  • Access team rosters and player statistics
  • Cricket news and updates section
  • Responsive design for mobile and desktop devices
  • Admin features to manage match schedules, teams, and news posts
