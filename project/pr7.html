Project Details

Project Name: Mouse Gaming Store

Description: This project is an e-commerce website that allows users to browse, purchase gaming mouse products, and manage their accounts. The system also provides an admin interface for managing categories, brands, and products.

Programming Languages/Tools: HTML, CSS, JavaScript, PHP, MySQL

Features:

  • Customer Features:
    • Register and login
    • Browse products and categories
    • View product details
    • Add products to the shopping cart
    • Checkout and payment using PayPal or offline methods
    • View order history
    • Submit feedback on the "Contact Us" page
  • Admin Features:
    • Manage categories, brands, and products (add, edit, delete)
    • View and manage customer orders
    • View user details and delete users if necessary
